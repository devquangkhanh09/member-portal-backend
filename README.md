# Member Portal project
- Author: team Software 1 - Big Data club.

## Description
- Portal for member management.

## Install dependencies

```bash
$ yarn install
```

## Run the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
